## ⚙️ 機能一覧

### 💰 通貨システム（v1.1.0〜）
- `/daily` … 20時間おきにログインボーナスを受け取る  
- `/balance` … 所持金を確認する  
- `/top type:balance` … 所持金ランキングを表示（データがない場合は「データがありません」と表示）

---

### 🎰 カジノ機能（v1.1.1〜）
- `/casino type: bet:`  
  - **type 候補:**  
    - `slot` … スロット  
    - `cointoss` … コイントス  

💡 `/casino` の実行結果により、勝敗に応じて通貨が増減します。
